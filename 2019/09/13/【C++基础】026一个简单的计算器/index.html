【C++基础】026一个简单的计算器

发表于 2019-09-13 分类于 C++基础

简介：编写一个简单的计算器。

【实例名称】

编写一个简单的计算器。

  • 编写一个简单的计算器功能，实现简单的加、减、乘、除表达式的计算。

  • 设用户输入的表达式具有如下格式：
    <操作数1>空格<运算符>空格<操作数2>
    如：3 + 42
    得：3 + 42 = 45

【问题分析】

  1. 用户输入表达式后，程序要判断是什么运算，然后再做相应的处理。
  2. 该问题可以使用if，或者if…else解决。
  3. 对于分支较多的情况，C++提供switch语句。switch语句也叫开关语句、多分支语句，它计算一个表达式的值，根据结果的不同，执行不同的分支处理语句。

【算法描述】

  1. 用num1，num2，op分别表示输入的表达式的是两个操作数和一个运算符。
  2. 如果op=‘+’，则result=num1 + num2，输出result；
  3. 如果op=‘-’，则result=num1 - num2，输出result；
  4. 如果op=‘*’，则result=num1 * num2，输出result；
  5. 如果op=‘/’，则
    • 如果num2=0，显示“除数为0”
    • 否则，计算result=num1 / num2，输出result；
  6. 其他，显示“运算符错误”。

    3+42=45

  3. Please input the expression:12 30
