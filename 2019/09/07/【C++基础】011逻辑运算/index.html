逻辑运算

1. 逻辑运算

  • 逻辑运算表达、判断多个条件之间的关系
    同时成立、之一成立、不成立
  • 例如
    • $x\in[0,1]$，可以表示为
      x >= 0 并且 x <= 1
    • $x\notin[0,1]$，可以表示为
      x > 1 或者 x < 0
  • 非负实数
    不是(x < 0)，非(x < 0)
  • “并且”，“或”，“不是”，就是逻辑运算

2. 逻辑运算符

表示逻辑运算的符号

  • &&——并且，逻辑“与”，and
  • ||——或，逻辑“或”，or
  • !——不是，逻辑“非”，not

3. 逻辑表达式

  • 逻辑运算符和关系表达式或有数值的表达式连接构成逻辑表达式
  • &&，||是双目运算，格式是
    <条件1> && <条件2>
    <条件1> || <条件2>
  • !是弹幕运算，格式是
    !<条件>

4. 逻辑表达式的值

  1. 逻辑值：true，false

  • 与运算 <条件1> && <条件2>

    条件1条件2结果
    true true true
    true false false
    false true false
    false false false

  • 或运算 <条件1> || <条件2>

    条件1条件2结果
    true true true
    true false true
    false true true
    false false false

  • 非运算 !

    条件结果
    true false
    false true

  • 举例

    逻辑表达式 -0.5 0.5 5
    x >= 0 && x <= 1< td> false true false
    x >= 0 || x <= 1< td> true false true
    !(x >= 0 && x <= 1)< td> true false true
    x < 0 true false false
    !(x < 0) false true true

6. 注意事项

  1. 单个变量或常量也可以作为条件
    1 && 2 ——— true
    1 && 0 ——— false
    0 || 1 ——— true
    0 || 0 ——— false
    !2 ——— false
    !0 ——— true
    a && b ——— a!=0 && b!=0
    a || b ——— a!=0 || b!=0
    !a ——— a==0，a为0时，!a为true
    a ——— a!=0
  2. 逻辑表达式的短路求值（short circuit evaluation）
    • 如果判断了前面的条件，已经能达到表达式的结果，就不再计算后面的条件
      如x>=0 && x<=1，如果x=-1，x>=0结果为false，那么此表达式的值必然为false，那么x<=1这里就不再计算
    • 逻辑运算符也用在条件分支语句当中
