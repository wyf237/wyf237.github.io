判断闰年并输出

【实例名称】

输入年份，判断该年是否闰年，是输出1，不是输出0

【例题分析】

  • 闰年的判断方法
    能被4整除且不能被100整除
    或能被400整除
  • 设年用y表示
    ((y%4 == 0) && (y%100 !=0)) || (y%400 == 0)

【算法设计】

设y表示年份，f是判断结果，是逻辑变量

  1. 输入y
  2. 计算
    f = ((y%4 == 0) && (y%100 !=0)) || (y%400 == 0)
    • 如果f为true，就是闰年
    • 如果f为false，就不是闰年
  3. 输出
    • cout << f << endl;
