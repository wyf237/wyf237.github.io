发表于 2019-09-07 更新于 2019-09-13 分类于 C++基础

加密解密

【实例名称】

  • 一段意义明确的文字，经过变换变得无法理解。合法用户可以将其变回原来的文字，而不合法的用户则不能，这就是加密。
  • 加密的一种方法叫做替换加密法，将原来文字中的字母用其后的第k个字母代替（26个字母的字母表看做是循环的，即z后面的字母是a）。
  • 例如，k=3，对“today”的每一个字母用其后的第3个字母代替，结果为“wrgdb”。
  • 编写程序，用户从键盘输入一个小写字母和k值，将输入的小写字母用替换加密法加密并输出加密后的字母。

【算法描述】

替换加密

  1. 输入单个小写字符c，和数k；
  2. 计算它在英文字母表中的序号n=c-‘a’（从0开始）；
  3. 将序号n加k（n=n+k）；
  4. 如果超出了25，则除26求余（n=n%26）
  5. 输出新序号位置的字母（ec=n+’a’）；
