【C++基础】028一维数组

发表于 2019-09-17 更新于 2019-09-22 分类于 C++基础

一维数组。

定义方法

  1. 语句格式
    <类型> <标识符>[<大小>];
  2. 举例
    • double length[30],width[30];
    • const int N=100,M=20;
    • int score[N*M]

初始化方法

  1. 语句格式
    <类型> <数组名>[<常量表达式>]={<表达式1>,<表达式2>，……}；
  2. 初始化方式
    • 全部元素
      double room[]={401,402,403,404,4051606,407,408,409,411};
    • 部分元素
      double average[30]={21.0,2012.7,7.5+2.3,a*b};

使用方法

  1. 语句格式
    <数组名>[<下标>]
  2. 举例

字符数组与字符串

  1. 语句格式
    sizeof(类型名)或者sizeof(变量名)
  2. 举例
    • char chr[]={‘H’,’e’,’l’,’l’,’o’}
      int a=sizeof(chr) //a=5
    • char chr[]=”Hello”
      int b=sizeof(chr) //b=6
