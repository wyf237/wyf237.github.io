整数、字符和字符串

1. 整数

  • 整数常量
    如：12，12L，0x15，015，12U
  • 整型变量
    int a,b;
    int x=10,y=20;

2. 字符

  • 字符常量
    如：’a’，’1’，’\t’，’\x41’，’\102’
  • 字符变量

3. 字符和整数

  1. 整数1，内存中存：
    00000000 00000000 00000000 00000001

  2. 字符’1’，内存中存：
    1这个字符的ASCII码，49
    0011 0001
    实际是个整数，占一个字节

  3. 字符和整数

    • 整数和字符在内存中存的都是整数。
    • 整数，存放数的二进制形式，字符，存放其ASCII码值。
    • 输出时，当整数看待，显示数的十进制形式。当字符看待，显示的是ASCII值是这个数的字符（该数<128）。

4. 字符串

  • 字符串常量
    “zhang”，”wang”，”Hello World”

  • 字符串变量

  • 内存中

    序号 012345...39
    char 'z''h''a''n''g''\0'...
    ASCII 122104971101030...
