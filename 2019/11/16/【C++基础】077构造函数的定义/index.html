【C++基础】077构造函数的定义

发表于 2019-11-16 分类于 C++基础

简介：构造函数的定义

OOP的封装

  1. 在类的定义中不难看出对实体数据的封装
    • 将数据成员和函数成员集成封装在“类体”重
    • 数据成员私有化，外界不能随便读写
    • 函数成员公有化，外界可以调用（运行）
    • 用类去生命各种对象，程序对对象进行相关处理
    • 核心是将实体设计成“统一体”去进行各种处理
  2. C++还提供了许多封装的方法
    • 构造方法
    • 析构方法

已设计出的类的共性

  1. 数据成员都是私有化成员，外界不能直接访问
  2. 都有一个成员函数，用来设置数据成员
  3. 该函数可以在需要时调用，达到随时修改数据成员的目的
  4. 数据成员值读出必须通过相关成员函数

构造函数的定义

  1. 在声明一个对象后，紧接着要给对象初始化
  2. 对象初始化实质上就是对所有数据成员赋值
  3. 如果对象中某个数据成员未赋值，则该数据成员的值不确定，那么该对象的值不完整
  4. 构造函数（Constructor）用于创建一个对象，提供了初始化该对象的一种简便手段
  5. 注意，在前面所有类的定义中都有一个成员函数完成初始化数据成员的功能，这个函数就可以挡成构造函数

  7. 注意事项：

    • 构造函数的函数名必须与类名相同
    • 构造函数没有返回值
    • 构造函数是将对象中所有数据成员进行初始化，一般由一系列赋值语句构成
    • 由系统在声明对象时自动调用
