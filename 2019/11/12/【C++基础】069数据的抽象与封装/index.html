如何抽象描述实体

从实体中抽取出若干特性和功能，来表示实体

  1. 移除细节看主干
  2. 不是借助具体形象反映现实，而是以抽象表达科学的真实

电视机的特征

  1. 型号
  2. 尺寸
  3. 材料
  4. 价格
  5. ……

  1. 播放影视
  2. 选频道
  3. 调节颜色

学生的特征

  1. 姓名

  3. 年龄

  6. ……

  1. 学习

抽象描述实体的结果不唯一！
无论怎么描述，都要尽可能抽象分析出主要特征与功能！
特征与功能密切相关、相互依赖，不能割裂！

什么是对象

对象食指被抽象描述的某个实体。

  1. 一个对象包含了实体的特征和功能
  2. 对象 = 实体的特征 + 实体的功能
  3. 对象 = 数据 + 函数（特征即为数据，功能即为函数）
  4. 将数据和函数当作一个统一体。

什么是类

具有相同特征和功能的对象集合。

  1. 所有红旗轿车→红旗轿车类。
  2. 所有小孩子→小孩类
  3. 所有公元日期→日期类
  4. 所有整数集合→整数类

对象是类的 一个实例 或 个体 或 元素
int k; k为整型变量，又称为整数对象。
struct date today; today称为日期结构体变量，又称日期对象

将实体转为程序中的数据（类型）

C++中有4种基本数据类型：字符、整数、浮点、双精度浮点。

当程序中涉及到的实体不能直接描述为基本数据类型时，只有描述成 新的数据（类型）。

类是相同属性的对象集合，其中某个对象的特征和功能就是类的特征和功能。

类与结构体

将数据和函数当作一个统一体——类
类是对可观世界万物的抽象描述

结构体是复合数据的统一体、

结构体与类的区别

  1. 结构体只有数据成员，没有函数成员
  2. 结构体又称为特殊的类
  3. 现在的结构体重也可以包括函数成员

类的语法格式与结构体的语法格式非常相似
