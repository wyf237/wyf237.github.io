【C++基础】050汉诺塔问题求解

发表于 2019-10-05 分类于 C++基础

简介：汉诺塔问题求解

【问题描述】

  • 有A，B，C三根柱子，在A柱子上有n个大小不同的金盘，大盘在下，小盘在上。
  • 要将A柱子上的金盘移动到C柱子上，每次只能搬动一个金盘，搬动过程中可以借助任何一根柱子暂时存放金盘，但必须满足大盘在下，小盘在上的条件。
  • 编程显示盘子移动的过程。n由用户输入。

【算法分析】

  • 如果只有一个盘子，只需一步，直接从A柱子移动到C柱，用A->C表示；
  • 如果有2个盘子，共需要移动3步：
    1. 把A柱子上的小盘子移动到B柱；用A->B表示；
    2. 把A柱子上的大盘子移动到C柱；用A->C表示；
    3. 把B柱子上的小盘子移动到C柱；用B->C表示；
  • 如果要将A柱子上的n个盘子（n值较大），移动到C柱子上去，必须先把上面的n-1个盘子从A柱子上移动到B柱子上暂存，按这种思路，就可以把n个盘子移动过程分作三大步：
    1. 把A柱子上的n-1个盘子移动到B柱；
    2. 把A柱子上剩下的盘子移动到C柱；
    3. 把B柱子上的n-1个盘子移动到C柱；
  • 其中n-1个盘子的移动过程又可以按同样的方法分为三大步，这样就把移动过程转化为一个递归过程，直到最后只剩下一个盘子，按照移动一个盘子的方法移动，递归结束。

【算法描述】

n个金盘汉诺塔问题

  • 函数hanoi(n,A,B,C)
    如果n=1，则
     显示A->C  //将一个金盘直接从A移到C
    否则
     hanoi(n-1,A,C,B)  //将n-1个金盘借助C从A移动到B
     显示A->C
     hanoi(n-1,B,A,C)  //将n-1个金盘借助A从B移动到C
  • 函数move
    将一个盘子从x移动到y
