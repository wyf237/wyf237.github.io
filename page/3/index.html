【C++基础】006-程序的基本要素

发表于 2019-08-31 更新于 2019-09-13 分类于 C++基础

标识符

  1. 程序中变量、类型、函数和标号的名称统称 标识符。

    • 如a，b，name，int，char，main，void等。

  2. 系统已有的标识符称为关键字。

  3. 常见关键字

    • using，namespace，void，return

    • int，float，double，char，bool，singed，unsigned，long，short，const，true，false，sizeof

    • if，else，for，do，while，switch，case，continue，break

    • class，struct，private，public，this，new，delete

  4. 自己给变量、函数起的名字称为自定义标识符

  5. 自定义标识符的命名规则

    • 由字母、数字和下划线组成。

    • 第1个符号不能是数字。

    • 标识符的长度：不同编译器规定的不同，但至少识别前6个字符，有的长度可达32个字符，甚至更长。

    • 区分大小写（大小写敏感）。

    • int a,A;这里a和A是2个不同的变量。
